## How to Edit Pages
1. Open any HTML file in a text editor.  
2. Change the text directly (look for paragraphs or headings).  
3. Save and upload again to GitHub.

---

### Home Page
Edit index.html  
To add a new project, copy one of the existing cards under “Featured Projects” and change the title, link, and image.

---

### Latest News
Edit index.html or news.html  
Find the section that says “Latest News”.  
Copy the example inside comments (it starts with <!-- Example -->) and fill in your own details.

Example:
<a class="card" href="news.html#spent-grain">
  <img src="images/blog_placeholder_01.jpg" alt="Spent grain update">
  <h3>Spent Grain Field Data Released</h3>
  <p class="lead">Early growth analysis of barley using brewery waste compost.</p>
</a>

---

### Project Pages
Each file inside /projects has a “Related News” section near the bottom.  
Copy one of the example cards there to add updates linked to that project.

---

### Community and Partners
community.html → add partner logos inside the logo-row  
partners.html → duplicate partner cards to add more collaborators

---

### Uploading Images
1. Place all images inside the /images/ folder.  
2. Reference them like this:
<img src="images/yourimage.jpg" alt="Description">

---

## Publishing Updates
1. Right-click the folder → “Compress” → thebiogreenproject.github.io.zip  
2. Go to your GitHub repository → Add file → Upload files  
3. Drag and drop the files or ZIP  
4. GitHub Pages updates automatically within 1-2 minutes.

---

### Quick Tips
- You can safely edit text directly in GitHub.  
- Use <!-- comments --> to leave notes.  
- Don’t worry if something breaks; GitHub keeps every version.
